# 🇧🇷 Versão em Português

## 🚀 Astronomy Picture of the Day (APOD)

Uma aplicação web simples que exibe a **Astronomy Picture of the Day (Imagem Astronômica do Dia)** utilizando a API oficial da NASA.

A aplicação busca diariamente uma imagem do espaço e mostra **título, data, imagem e explicação** diretamente na página.

### 🌌 Funcionalidades

* Consome dados da **API APOD da NASA**
* Exibe:

  * Imagem do dia
  * Título
  * Data
  * Explicação da NASA
* Tratamento de erro caso a requisição falhe
* Layout simples e responsivo
* Renderização dinâmica com JavaScript

### 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* NASA APOD API

### ▶ Como Executar

1. Clone o repositório

```
git clone https://github.com/andrezon04/nasa-data-fetcher.git
```

2. Abra a pasta do projeto

3. Abra **index.html** no navegador

# 🇺🇸 English Version

## 🚀 Astronomy Picture of the Day (APOD)

A simple web application that displays NASA's **Astronomy Picture of the Day** using the official NASA API.

The application fetches daily space images and displays the **title, date, image, and explanation**.

### 🌌 Features

* Fetches data from the **NASA APOD API**
* Displays:

  * Image of the day
  * Title
  * Date
  * Explanation
* Error handling if the request fails
* Clean and responsive layout
* Dynamic rendering with JavaScript

### 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* NASA APOD API

### ▶ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/apod-project.git
```

2. Open the project folder

3. Open **index.html** in your browser
