## 🚀 Astronomy Picture of the Day (APOD)

A simple web application that displays NASA's **Astronomy Picture of the Day** using the official NASA API.

The application fetches daily space images and displays the **title, date, image, and explanation**.

### 🌌 Features

* Fetches data from the **NASA APOD API**
* Displays:

  * Image of the day
  * Title
  * Date
  * Explanation
* Error handling if the request fails
* Clean and responsive layout
* Dynamic rendering with JavaScript

### 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* NASA APOD API

### ▶ How to Run

1. Clone the repository

```
git clone https://github.com/andrezon04/nasa-data-fetcher.git
```

2. Open the project folder

3. Open **index.html** in your browser
